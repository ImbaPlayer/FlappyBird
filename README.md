# FlappyBird
### 简介
这是一个使用 [Qt](https://www.qt.io/) 编写的Flappy Bird游戏。
### 环境
* Qt 5.0 or higher
* Windows / Linux

### 使用方法
* 使用Qt Creator编译项目即可
